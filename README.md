
# Coin Toss App

A small React experience that lets you flip a virtual coin, review the most recent results, and watch the odds evolve with every toss.

## Getting started

```bash
git clone https://github.com/sunnymendapara15/coin-toss-app.git
cd coin-toss-app
npm install
npm start
```

The development server starts on [http://localhost:3000](http://localhost:3000). Any code changes you make will reload the page automatically.

## Available scripts

- `npm start` — Starts the development server with hot reloading.
- `npm run build` — Bundles the app for production into the `build/` folder.
- `npm test` — Launches the test runner in watch mode.

## Features

- A single button to toss a virtual coin (Heads/Tails).
- Live stats for how many times each side has appeared.
- A history of the five most recent tosses and the ability to reset the session.
- A 3D metallic coin with animated glow, dramatic drop shadow, and responsive shimmy.
- A subtle metallic chime (Web Audio API) that plays with every toss.
